# 12 — Contributing and community

## Quick start

See **[CONTRIBUTING.md](../CONTRIBUTING.md)** for environment setup (Node, pnpm, Rust/Tauri) and PR expectations.

## Code of conduct

**[CODE_OF_CONDUCT.md](../CODE_OF_CONDUCT.md)** applies to all project spaces.

## Documentation changes

- **`docs/`** is maintained in **English** for contributors and AI tooling.
- Runtime or schema edits must stay synchronized with [04-data-model-schema.md](./04-data-model-schema.md), JSON Schema, tests, and ADRs.

## Issues

When `.github/issues/` templates exist, prefer them for **Vertical Slice 1** work items; the task narrative lives in [14-vertical-slice-1-tasks.md](./14-vertical-slice-1-tasks.md).
